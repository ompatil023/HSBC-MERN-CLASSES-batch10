 Tokens: 
 -------
    ->smallest units in the Javascript programming
    ->Tokens are classified into 6 types 
        1. identifiers
            ->identifiers are the names give JS code 
            ->in the place of identifiers we can not use keywords
            ->Ex: variable name , class names , function names 

            Rules : 
                ->identifiers not starts with numbers , special characters instated of _ ,$ 
                -identifiers doesn't have spaces

            
        2. keywords 
                ->keywords predefine words present js programming 
                ->In js total 66 keywords are present 
                Ex: class , const , let, var , if, else , while , do, switch , break , case , continue ..etc

        3. literals 
            ->literals are values given to Js Code 
            Ex: 
            String literals             str="java"
            Numeric literals            num=800



        4. separators
            ->separators are used to separate give JS code 
            Ex: " " ,'' ,`` , ; : , . { } [ ] ( )

        5. operators    
            ->operators are the symbols use to perform some operations on operands 
            Ex: 10 + 20
            Types: 
            -------
                1. Athematic operators (+,-,/,%, * ,**)
                2. logical operators (&& ,|| ,!)
                3. bitwise operators (&,|,^)
                4. assignment operators (= ,+= ,-= ,/=,%= ,*=)
                5. relational operators (>,>= ,< ,<=)
                6. comparison operators (==,=== ,!=)
                7. unary operators  (expr-- , expr++ , --expr , ++expr)
                8. ternary operators (condition ? expr1 : expr2)
                9. shift operators (>>,<<)


        6. comments 
            ->comments are used to comment given code 
            ->single line comments , multi line comments


Variables: 
------------
    ->Variables are memory location used to store data/ values and it can change N of time during execution


    i. Variable declaration 
    ii. variable Initialization 
    iii.  variable utilization 
    iv. Variable Re-declaration 
    vi. variable Re-Initialization 


    1.let   
        ->let Variables act's like a local/block scope variable
        ->we declare and Initialize let Variables in same line/ different lines is possible
        -> Re-Initialization is possible but Re-declaration not possible
        ->we can't hoist let Variables


    2.var 
        ->var Variables act's like a global  variables
        ->we declare and Initialize var Variables in same line/ different lines is possible
        -> Re-Initialization and  Re-declaration is possible 
        ->we can hoist var Variables
        
    3.const 
        ->const  Variables act's like a final variable
        ->we declare and Initialize const  Variables in same line is possible but in  different lines is not possible
        -> Re-Initialization and Re-declaration not possible
        ->we can't hoist const Variables
     





hoisting : 
----------
    ->utilizing the variable before declaration 
    ->hoisting is possible only in var variables
