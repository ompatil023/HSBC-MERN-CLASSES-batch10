Components in React js : 
---------------------------
    ->smallest UI(user interface ) blocks in the Screen/Webpage

    ->Types  : 
        1. Class Based Component (CBC)
            ->any Components declare using class keywords 
            ->CBC is also called as stateful Components
            ->CBC states are present 
            ->super keyword , this keyword and constructor is present 
            ->life cycle methods present


        2. function Based Component(FBC)
            ->any Components declare using function keywords 
            ->FBC is also called as stateless Components
            ->FBC states are not present 
            ->super keyword , this keyword and constructor is  not present 
            ->life cycle methods also not  present
